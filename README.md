# sky widget bundle for Tesserae

Aurora alert score, current moon phase, overhead air traffic, and Australian BOM weather warnings.

Drop into [Tesserae](https://github.com/dmellok/tesserae) via Settings → Widgets → Browse community widgets.

## Folders shipped

- `sky_air_traffic`
- `sky_aurora`
- `sky_bom_warnings`
- `sky_moon`
